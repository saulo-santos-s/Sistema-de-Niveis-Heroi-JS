# Sistema de Níveis de Herói em JavaScript

Este projeto é um simples programa em **JavaScript** que define o nível de um herói baseado na sua **experiência (XP)**. É uma ótima forma de praticar estruturas condicionais (`if`, `else if`, `else`) e concatenar strings.

## Funcionalidades

- Define o nome do herói.
- Atribui pontos de experiência (XP).
- Determina o nível do herói com base no XP:
  - Ferro: XP < 1000
  - Bronze: 1000 ≤ XP < 2000
  - Prata: 2000 ≤ XP < 5000
  - Ouro: 5000 ≤ XP < 7000
  - Platina: 7000 ≤ XP < 8000
  - Ascendente: 8000 ≤ XP < 9000
  - Imortal: 9000 ≤ XP < 10000
  - Radiante: XP ≥ 10000
- Mostra no console uma mensagem com o nome do herói e seu nível.
